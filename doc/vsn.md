vsn
=========
Display the release vsn.

## Overview
Examine the current release version from `start_erl.data`.  
If the release version of program running on ErlangVM is in contradiction with `start_erl.data`,
there is a possibility to return the actual and different version.

## Usage

```
$ rebar3 help erlup vsn   # using rebar3.
$ erlup vsn -h            # using escript.
```
